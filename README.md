# Live Operational Impact Dashboard — Non-Profit Infrastructure

## 📌 Project Overview
This project focuses on transforming over 10,000 rows of fragmented field entries into a unified, high-performing data model. The end product is an interactive Power BI operational tracking tool designed explicitly for executive leadership to monitor key regional metrics and performance totals in real-time.

Instead of highlighting technical data cleaning anomalies, this repository showcases the final live operational impact, providing leadership with a clear, strategic view of organizational metrics to streamline executive decision-making.

---

## 🛠️ Tech Stack & Skills Demonstrated
* **Data Modeling:** Combined multiple fragmented field entries into a single, cohesive schema.
* **Business Intelligence:** Deployed an operational dashboard using Power BI Desktop.
* **Analytical Focus:** Designed for executive leadership, focusing heavily on location-based performance and operational totals.

---

## 📊 Key Features of the Dashboard
* **Executive Summary View:** High-level operational totals visible at a single glance for rapid decision-making.
* **Location Analytics:** Regional performance tracking to identify high-impact areas and resource distribution.
* **Live Tracking Framework:** Structured to handle continuous data entry while maintaining data integrity across the model.
